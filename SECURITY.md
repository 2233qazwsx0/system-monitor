# Security

## Reporting a Vulnerability

If you find a security issue in System Monitor, please open a **private** security advisory on GitHub rather than a public issue.

- **Repository**: https://github.com/[YOUR_USERNAME]/system-monitor/security/advisories
- **Response time**: within 72 hours

Please include: steps to reproduce, affected version, platform, and any logs.

## Automated Security

We use Dependabot (see `dependabot.yml`) to keep dependencies up to date.
Every commit is scanned with CodeQL (see `.github/workflows/codeql.yml`).
